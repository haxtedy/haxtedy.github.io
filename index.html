前端的一些面试点

1.输入完网址按下回车，到看到网页这个过程中发生了什么？

  • 域名解析：DNS（Domain Name Server，域名服务器）将域名解析为对应的IP地址。
  • 发起TCP的3次握手，建立客户端与服务器的连接。
  • 客户端发起http请求。
  • 服务器响应http请求，返回请求页面的html代码。
  • 浏览器得到html代码后，解析html代码，并请求html代码中的资源。
  • 服务器再次返回请求资源。
  • 浏览器得到所需的所有资源，对页面进行渲染，呈现给用户。

HTTP_CDN

CDN概述

CDN的全称是Content Delivery Network，即内容分发网络。CDN是构建在网络之上的内容分发网络，依靠部署在各地的边缘服务器，通过中心平台的负载均衡、内容分发、调度等功能模块，使用户就近获取所需内容，降低网络拥塞，提高用户访问响应速度和命中率。CDN的关键技术主要有内容存储和分发技术。

CDN的优势：

  • CDN节点解决了跨运营商和跨地域访问的问题，访问延时大大降低；
  • 大部分请求在CDN边缘节点完成，CDN起到了分流作用，减轻了源站的负载。

JS作用域

js中的作用域分为全局作用域和局部作用域(方法作用域)。在ES6中新增了块级作用域。

全局作用域

var定义的变量属于局部作用域，所以在 if 或是 for 这样逻辑语句中使用var定义的变量，都是可以在其所属的作用域中访问的。以下代码，a和b实际上都是定义在全局作用域中，此处var的范围扩展到了全局。

局部作用域

局部作用域也可以称之为函数作用域、方法作用域。

  • 局部作用域中定义的变量，只供局部作用域调用，外界无法访问。

  • 在局部作用域中，可以访问其父级作用域中的变量（不包括父级的this和arguments）。这里涉及了作用域链。

块级作用域

块作用域由 { } 包括，if语句和for语句里面的{ }也属于块作用域。let和const定义的变量属于块级作用域。

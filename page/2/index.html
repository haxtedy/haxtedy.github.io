linux_command

  1. ls：查看当前目录的信息。常用参数 -l :查看文件详细信息, -a ：查看当前目录所有文件，包括隐藏文件 .file。可连打 ls -al。

  2. cd：全称是change directory，变换目录，不能进入文件。接受相对路径或者绝对路径的目录作为参数。cd .. 代表上级目录，cd / 斜杠代表的根目录。

  3. mkdir：创建目录。参数 -p 为递归的创建目录,如 mkdir -p /lina/music。

  4. mv 移动或者重命名文件(move)。

    • mv 文件名 文件名 将源文件名改为目标文件名
    • mv 文件名 目录名 将文件移动到目标目录
    • mv 目录名 目录名 目标目录已存在，将源目录移动到目标目录；目标目录不存在则改名
    • mv 目录名 文件名 出错

  5. touch：接收pathname作为参数，创建空白文件。

  6. cp：copy，复制文件或者文件夹，使用方式为：cp old_pathname new_pathname 加 r参数复制文件夹，f参数强制复制，覆盖，cp -rf 就是强制复制文件夹了。

  7. scp：主要是在不同的Linux系统之间来回copy文件 。

    • 从 本地 复制到 远程 scp /home/daisy/full.tar.gz root@172.19.2.75:/home/root
    • 从 远程 复制到 本地 scp root@/172.19.2.75:/home/root/full.tar.gz /home/daisy/

  8. rm：删除文件，或文件夹，rm pathname 删除文件，-r参数删除文件夹，-f参数强制删除，慎用切记一定慎用，最好别用rm -rf。

  9. rmdir： 等同于 rm -r 不过rmdir只能删除空白文件夹。

  10. cat：是linux下的一个文本输出命令。

    • 一次显示整个文件。 cat filename
    • 从键盘创建一个文件，只能创建新文件,不能编辑已有文件。 cat > filename
    • 将几个文件合并为一个文件。 cat file1 file2 > file

  11. vim：vi是Linux中最基本的文本编辑器,vim就是vi的升级版，是所有Unix及Linux系统下标准的编辑器。

  12. ln：用于将一个文件创建链接,链接分为软链接(类似于windows系统中的快捷方式)和硬链接(相当于对源文件copy，程序或命令对该文件block的另一个访问路口)，命令默认使用硬链接。

    语法：ln [选项] 源文件 链接文件
    选项：-s 对源文件创建软链接

    ln -s 源文件 符号链接文件名 ln -s python3.5 python 这样python就是python3.5的一个符号链接文件了。

    需要多看看实际是怎么用的。

  13. man：查询手册，获得帮助信息。

    • 1 普通命令 man 1 ls
    • 2 系统调用
    • 3 库函数

  14. find：用来在文件系统中，查找一个文件所在。如：find 目录 -name 文件名 find /etc -name “interfaces”

  15. ps：进程查看命令，可以确定有哪些进程正在运行和运行状态。它显示的是进程某一时刻的状态，并不连续。如果想对进程实时监控，应该用top命令。

    • -a ： 显示现行终端机下的所有进程，包括其他用户的进程；
    • x ：通常与 a 这个参数一起使用，可列出较完整信息。
